#  OJT Progress Tracker

A simple and interactive web application to track daily learning tasks during On-the-Job Training (OJT).

---

##  Features

- Add tasks with title, description, category and date
- Edit tasks (title + description)
- Delete tasks with confirmation
- Mark tasks as completed or pending
- Filter tasks (All / Completed / Pending)
- Sort tasks by date
- View summary (Total, Completed, Pending)
- Data stored using Local Storage
- Toast notifications for actions

---

##  Tech Stack

- HTML
- CSS
- JavaScript (Vanilla JS)
- Local Storage

---

##  How to Run

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser
