#  OJT Progress Tracker

A modern and responsive web application to track daily tasks, learning progress, and project activities during On-the-Job Training (OJT).

---

##  Features

-  Add new tasks with title, date, category, and description  
-  Edit existing tasks  
-  Mark tasks as completed / pending  
-  Delete tasks  
-  Dashboard showing total, completed, and pending tasks  
-  Filter tasks (All / Completed / Pending)  
-  Sort tasks by date (Newest / Oldest) or alphabetically  
-  Dark Mode support  
-  Data stored in browser (LocalStorage)  
-  Fully responsive (Mobile + Desktop)

---

##  Tech Stack

- HTML5  
- CSS3 (Custom UI, no frameworks)  
- JavaScript (Vanilla JS)  
- LocalStorage (for persistence)
